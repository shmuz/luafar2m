What's this
===========
Integration between far2m and mpv, allowing to use mpv to transmit currently
displayed file to Far, and select by pressing Insert key.


How to use
==========
copy ipc.lua to ~/.config/far2m/Macros/scripts/
copy mpv-far2m-ipc.lua to ~/.config/mpv/scripts/
